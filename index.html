Sobre mim

Me chamo João Gabriel Rocco e sou estudante de Ciência da Computação, atualmente no primeiro período. Sempre fui muito interado no mundo da tecnologia.

Estou dando meus primeiros passos no mundo da programação e desenvolvimento, aprendendo linguagens como Python, HTML, CSS e explorando conceitos fundamentais de lógica, algoritmos e desenvolvimento web.

Este portfólio tem como objetivo acompanhar minha evolução, compartilhar meus projetos e aprendizados ao longo da minha trajetória acadêmica e profissional.

Meus Projetos

Projeto 1

Jogo no Construct

Jogo de plataforma 2D desenvolvido no Construct, com desafios, obstáculos e inimigos. Explora lógica de programação, mecânicas de jogo e design de fases.

Projeto 2

Simulador de Reciclagem no Processing

Simulador desenvolvido no Processing que ensina sobre reciclagem, onde o usuário arrasta os resíduos para as lixeiras corretas, promovendo educação ambiental.

Projeto 3

Site interativo desenvolvido para exibir pontos sustentáveis da cidade de Curitiba. O mapa mostra locais ecológicos, parques e outras iniciativas sustentáveis, utilizando tecnologias web e APIs de mapas.

Projeto 4

Projeto de um carrinho irrigador utilizando Arduino, começamos a base do projeto no tinkercad.
